# Core Java

This repository contains practice programs covering fundamental to intermediate Core Java concepts. Each program focuses on an individual topic and demonstrates how Java features work in real scenarios.

The repository is designed to strengthen Java fundamentals, prepare for backend development, and improve understanding of object-oriented programming along with commonly used Java APIs.

---

## Topics Covered

### 1. Java Basics
- Introduction to Java
- JVM, JRE, and JDK
- First Java program
- Compiler and JVM execution process
- Classes and Objects
- Static members
- Instance and private members
- Wrapper Classes
- Command Line Arguments
- Packages
- Access Specifiers

### 2. Constructors and OOP Concepts
- Constructors
- Types of Constructors
- Constructor chaining
- `this` keyword
- `super` keyword
- Inheritance
- Types of Inheritance
- Polymorphism
- Method Overloading
- Method Overriding
- Abstract Classes
- Interfaces
- Abstract Class vs Interface

### 3. Multithreading
- Introduction to multithreading
- Thread lifecycle
- Thread implementation techniques
- Thread priorities
- Synchronization basics

### 4. Exception Handling
- Exception hierarchy
- try, catch, finally
- throw and throws
- Checked and Unchecked exceptions
- Practical programs on exception handling

### 5. File Handling
- File class
- Reading files
- Writing files
- FileInputStream and FileOutputStream

### 6. Collections Framework
- Introduction to Collections
- Collection interface
- List, Set, Queue
- ArrayList
- LinkedList
- Vector
- Stack
- Collections utility class
- Enumerator
- Iterator
- ListIterator

---

## Project Structure

This repository does not follow a folder-based structure.  
All Java files are kept directly inside the main directory for quick access and practice convenience.

---

## Requirements
- JDK 8 or higher

## How to Run

Compile:
```bash
javac FileName.java
```

Run:
```bash
java FileName
```
